C# — Scope e Blocchi
Blocchi dichiarativi vs blocchi di esecuzione, inizializzatori, regola mentale
Regola Mentaleconcetto
Scopo dichiarativo → solo dichiarazioni, zero codice eseguibile
I blocchi di namespace, class, interface, struct contengono solo dichiarazioni di membri: campi, proprietà, metodi, eventi. Non puoi scrivere Console.WriteLine() o assegnazioni fuori da un metodo.
Scopo di esecuzione → codice eseguibile permesso
I metodi, costruttori, property body, lambda, blocchi if/for/while eseguono logica. Qui va il codice di business.
1. Blocchi Dichiarativi — Solo Dichiarazionideclarative
namespace N
{
    class C
    {
        // ✅ Solo dichiarazioni di membri
        int x;
        string name;
        public void Metodo() { }

        // ❌ Questo NON è valido qui:
        // Console.WriteLine("errore");
        // x = 5;
    }
}

interface I
{
    // Solo firme di metodi/proprietà
    void Esegui();
    string Nome { get; }
}

struct S
{
    // Solo dichiarazioni
    public int X;
    public int Y;
}
2. Blocchi di Esecuzione — Logica Eseguibileexecution
void Metodo()
{
    // ✅ Istruzioni eseguibili
    Console.WriteLine("Esempio");
    int x = 5;
}

MyClass()  // costruttore
{
    // ✅ Istruzioni eseguibili
    _service = new Service();
}

if (condizione)
{
    // ✅ Istruzioni eseguibili
    DoSomething();
}

for (int i = 0; i < 10; i++)
{
    // ✅ Istruzioni eseguibili
    Process(i);
}
3. Inizializzatori — Eccezione Consentita nella Classeinizializzatori
public class C
{
    // ✅ Inizializzatori: consentiti direttamente nel corpo classe
    public int a = 10;

    public List<int> nums = new() { 1, 2, 3 };

    public MyService _svc = new MyService();

    // ❌ Chiamata a metodo NON è un inizializzatore valido:
    // Console.WriteLine(a);  // errore di compilazione
}
Perché funzionano gli inizializzatori?
Il compilatore C# sposta automaticamente gli inizializzatori di campo nel costruttore. Quindi public int a = 10 è zucchero sintattico per this.a = 10 dentro il costruttore implicito.
Errori Tipici da Neofitierrori
public class MyService
{
    // ❌ SBAGLIATO — chiamata a metodo fuori da un blocco di esecuzione
    ValidateConfig();   // errore CS0106

    // ❌ SBAGLIATO — assegnazione semplice (non inizializzatore)
    x = GetDefaultX();  // errore

    // ✅ CORRETTO — metti la logica nel costruttore
    public MyService()
    {
        ValidateConfig();
        x = GetDefaultX();
    }
}
