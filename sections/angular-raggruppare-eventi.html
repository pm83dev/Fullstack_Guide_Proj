Angular — Raggruppare e Accoppiare Eventi
Pattern pairEvents() con Map, accoppiamento START/STOP, calcolo durata, template con badge
Concetto — Accoppiare START e STOPpattern
Problema
Gli eventi arrivano come array piatto. Ogni evento ha un eventGroupId che identifica la coppia START/STOP. L'obiettivo è trasformare l'array in coppie, calcolare la durata, e mostrare "In Corso" se lo STOP manca.
pairEvents() — Funzione di Raggruppamento con Maptypescript
private pairEvents(events: EventDataModel[]): EventPair[] {
  const pairs: EventPair[] = [];
  const grouped = new Map<string, EventDataModel[]>();

  // 1. Raggruppa per eventGroupId (fallback su eventId)
  events.forEach(event => {
    const groupId = event.eventGroupId || event.eventId;
    if (!grouped.has(groupId)) grouped.set(groupId, []);
    grouped.get(groupId)!.push(event);
  });

  // 2. Per ogni gruppo cerca START e STOP
  grouped.forEach((groupEvents, groupId) => {
    const startEvent = groupEvents.find(e =>
      e.eventType?.includes('START') || e.eventType?.includes('AVVIAMENTO')
    );
    const stopEvent = groupEvents.find(e =>
      e.eventType?.includes('STOP') || e.eventType?.includes('FINE')
    );

    // 3. Calcola durata se entrambi presenti
    let duration: number | undefined;
    let durationFormatted: string | undefined;

    if (startEvent?.eventDT && stopEvent?.eventDT) {
      const start = new Date(startEvent.eventDT);
      const stop  = new Date(stopEvent.eventDT);
      duration = stop.getTime() - start.getTime();
      durationFormatted = this.formatDuration(duration);
    }

    pairs.push({ groupId, startEvent, stopEvent, duration, durationFormatted });
  });

  return pairs;
}
Interfacce TypeScriptmodels
interface EventPair {
  groupId: string;
  startEvent?: EventDataModel;
  stopEvent?: EventDataModel;
  duration?: number;         // millisecondi
  durationFormatted?: string; // es. "1h 23m 45s"
}

interface EventDataModel {
  eventId: string;
  eventGroupId?: string;
  eventType?: string;   // es. 'START_LAVORAZIONE', 'STOP_LAVORAZIONE'
  eventDT?: string;     // ISO date string
  articleRef?: string;
  nrOrdine?: string;
  pzStampati?: number;
  pzTagliati?: number;
}

Punti Chiave del Patternbest practice
Map<string, T[]> per raggruppamento
Usa new Map() invece di un oggetto plain per raggruppare: has(), get(), set() sono type-safe e preservano l'ordine di inserimento.
Evento senza coppia = "In Corso"
Se stopEvent è undefined, la lavorazione è ancora in corso. Mostra il badge giallo invece della durata.
Fallback su eventId
event.eventGroupId || event.eventId — se l'evento non ha un groupId, si usa l'eventId stesso come chiave. In questo caso risulterà sempre una coppia con un solo evento.
