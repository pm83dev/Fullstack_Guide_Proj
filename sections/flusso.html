Flusso Completo NGXS
Service → Action → State → Component: la pipeline intera in un colpo d'occhio
Component → store.dispatch(new GetProductsAction(...))
↓
State → @Action intercetta, chiama service
↓
Service → HTTP GET → ritorna Observable<IModel>
↓
State → tap() → ctx.patchState({ products: result.data })
↓
Component → @Select(ProductState.products) products$ → async pipe
