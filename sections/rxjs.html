I concetti minimi da tenere in testa per non perdersi
Glossario rapido
ConcettoSpiegazione
ObservableRisultato che arriverà dopo (flusso asincrono)
subscribe()Cosa fare quando il risultato arriva
tap()Effetto collaterale senza modificare il valore (es: patchState)
catchError()Gestione errori nel pipe
throwError()Ripropaga l'errore dopo catchError
pipe()Catena di operatori sull'Observable
async pipeGestisce subscribe/unsubscribe automaticamente nel template
Pattern con subscribe nel component
this.store.dispatch(new InsertOrderAction(payload))
  .subscribe({
    next: () => {
      // successo: svuota carrello, apri modale
      this.clearCart();
      this.showSuccessModal = true;
    },
    error: (err) => {
      // errore: mostra messaggio
      this.errorMessage = 'Errore nell\'ordine';
    },
  });
