Action
Classi che rappresentano eventi o comandi. Il payload viaggia nel costruttore.
product.action.ts action 
// product.action.ts
export class GetProductsAction {
  static readonly type = '[Product] Get Products';
  constructor(public payload?: any) {}
}

export class GetProductByIdAction {
  static readonly type = '[Product] Get Product By ID';
  constructor(public id: number) {}
}

// Auth example
export class LoginSuccess {
  static readonly type = '[Auth] Login Success';
  constructor(public user: User, public token: string) {}
}
💡 Convenzione naming

Il type segue il pattern [Contesto] Verbo Oggetto. Es: [Product] Get Products, [Auth] Login Success. Serve per il DevTools di NGXS.
