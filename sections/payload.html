action.payload
Come trasportare dati dal componente allo State — e tra azioni
Direzione del flusso
Component → dispatch(new CreateUser(userData)) — payload parte da qui
↓
Action → constructor(public payload: CreateUserDto) {}
↓
State → action.payload — ricevuto, usato per chiamare il service
↓
Service/API → Risposta → nuova Action di successo con NUOVO payload
Definizione azioni con payload
// Payload singolo tipizzato
export class CreateUser {
  static readonly type = '[User Form] Create User';
  constructor(public payload: CreateUserDto) {}
}

// Payload multiplo (id + dati)
export class UpdateUser {
  static readonly type = '[User Form] Update User';
  constructor(public id: string, public payload: UpdateUserDto) {}
}

// Payload semplice
export class DeleteUser {
  static readonly type = '[User List] Delete User';
  constructor(public userId: string) {}
}
Dispatch dal componente
createUser() {
  const newUser: CreateUserDto = { name: 'Mario', email: 'mario@example.com' };
  this.store.dispatch(new CreateUser(newUser));
}

updateUser(id: string) {
  this.store.dispatch(new UpdateUser(id, { name: 'Mario Updated' }));
}

// Dispatch condizionale (add vs edit)
saveUser(userData: any) {
  const action = userData.id
    ? new UpdateUser(userData.id, userData)
    : new CreateUser(userData);
  this.store.dispatch(action);
}
Accesso al payload nello State
// Payload singolo
@Action(CreateUser)
createUser(ctx: StateContext<UserStateModel>, action: CreateUser) {
  const newUserData = action.payload;       // type: CreateUserDto
  return this.userService.createUser(newUserData).pipe(...);
}

// Payload multiplo con destrutturazione
@Action(UpdateUser)
updateUser(ctx: StateContext<UserStateModel>, action: UpdateUser) {
  const { id, payload: updateData } = action;
  return this.userService.updateUser(id, updateData).pipe(...);
}

// Payload complesso con destrutturazione
@Action(LoadUsers)
loadUsers(ctx: StateContext<UserStateModel>, action: LoadUsers) {
  const { page, pageSize, filters } = action.payload;
  ctx.patchState({ loading: true });
  return this.userService.getUsers(page, pageSize, filters).pipe(...);
}
